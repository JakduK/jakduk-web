#jakduk-web
Single page application

##Server
* Node.js 6.9.1+
* Express
* Environment variables
  * PORT (Default: 3000)
  * NODE_ENV (Default : development)
##Client
* Vue.js
* semantic-ui

#Prerequisite
```
npm install -g bower gulp webpack
```

#Install
```
bower install
npm install
webpack
```

#Run
```
NODE_ENV=production npm start
NODE_ENV=development npm start
```
