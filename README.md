#jakduk-web

#Environment
* Node.js 6.9.1+
* npm 3.10.8+
* Default PORT : 3000
* Default NODE_ENV : development

#Prerequisite
```
npm install -g bower gulp
```

#Install
```
bower install
npm install
cd client/semantic && gulp build
```

#Run
```
NODE_ENV=production npm start
NODE_ENV=development npm start
```
