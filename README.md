# jakduk-web
* Single page application
* Stage https://dev-web.jakduk.com
* Production https://www.jakduk.com

## Client
* Vue.js
* semantic-ui

## Server
* Node.js 10.17.0+
* Express
* Environment variables
  * PORT (Default: 3000)
  * NODE_ENV (Default : development)

# Install
```
npm install
```

# Run
```
NODE_ENV=production npm start
NODE_ENV=development npm start
```
